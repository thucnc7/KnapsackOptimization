👋 Chào mừng đến với Knapsack Visualizer

Webapp này có 8 chế độ chạy. Bấm vào tên để biết mode đó làm gì:

📊
Dashboard
Tổng quan CSV benchmark — success/timeout, time/memory theo thuật toán, biểu đồ thời gian theo N
⚖️
So sánh
Chọn 2 thuật toán bất kỳ → scatter log-log, speedup theo N, value khớp/lệch
▶️
Single Solver
Chạy 1 thuật toán live trên 1 instance với timeout cấu hình được. Hiển thị banner đếm thời gian + items được chọn
🚀
Multi-Solver
Streaming nhiều thuật toán cùng 1 instance, status update realtime mỗi 250ms
🛠️
Custom Instance
Builder thủ công hoặc sinh ngẫu nhiên với config: n, max_weight, capacity_ratio, Pearson r
🔁
Sensitivity
Phân tích warm-start Dual Simplex — chứng minh tại sao có Dual Simplex (speedup hàng nghìn lần)
📈
Data Quality
Test scenarios config + biểu đồ phân phối dữ liệu sinh ra
⚙️
Job Runner
Chạy nền benchmark, generator, quality.py với log streaming + progress bar

Sensitivity Analysis — Warm-start Dual Simplex

Khi instance bị thay đổi nhỏ (capacity, value, hoặc thêm constraint mới), Dual Simplex có thể warm-start từ optimal tableau cũ thay vì giải lại từ đầu. Phân tích này so sánh wall-time + iterations giữa 2 phương án resolve-from-scratch và warm-start trên 600 runs (200 instances × 3 scenarios).
